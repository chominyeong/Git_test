## 깃허브 레포지토리 복제
1. 깃허브 레포 생성 → 주소 복제

2. vscode 레포지토리 복제 클릭 → 주소 넣기

3. 복제된 레포 폴더 어디에 설정할 건지 선택

## 커밋, 푸시
1. 파일 생성됨

2. 왼쪽 세번째 로고(동글뱅이3개연결된거) 클릭

3. 레포지토리에 넣을 파일 +클릭 → -(스테이징된변경사항)로 바뀔거임

4. 메시지(Ctrl+Enter(으)로 "main"에 커밋)에다가 커밋 메시지 작성 → 아래 체크 커밋 박스 클릭

5. 아래 커밋이 몇 개 가능한지 보이는 그림 클릭(푸시)
<p align="center"><img src="https://ifh.cc/g/9gcrct.png" width="300" height="750"/></p>

6. 확인 클릭
<p align="center"><img src="https://ifh.cc/g/MLr19Y.png" width="400" height="120"/></p>

7. 깃허브에서 푸시가 되었는지 확인하기!!

## 브랜치를 새로 만들어 커밋, 푸시

<img src="https://ifh.cc/g/ZzKAtX.png" width="400" height="100"/>

3. 파일 수정

4. 커밋 메시지 작성하고 커밋하면 브랜치에 생성됨.

<p align="center"><img src="https://ifh.cc/g/TPoZHd.png" width="300" height="280"/></p>
